# Personas — team-tasks

> MVP 헌장: 단일 테이블 · 기능 5개 이하 · 1일 안에 코드로 옮길 수 있는 분량

| 항목 | 팀 리더 (Task Assigner) | 팀원 (Task Doer) | 부서장 (Observer) |
|------|------------------------|-----------------|------------------|
| **역할** | 업무를 만들고 팀원에게 배정하는 사람 | 배정된 업무를 수행하고 상태를 갱신하는 사람 | 팀 전체 진행 현황을 조회하고 보고하는 사람 |
| **기술 수준** | 중급 — 웹 앱 일상적으로 사용, 단축키·필터 등 고급 기능도 금방 익힘 | 초급 — 모바일 위주, 화면이 복잡하면 이탈. 클릭 3회 이내 완료를 선호 | 초급-중급 — 주로 읽기 전용으로 사용, 수치 요약·차트 선호 |
| **주요 시나리오 1** | 새 업무 생성 후 담당자·마감일·우선순위를 한 화면에서 설정한다 | 오늘 내 할 일 목록을 열어 우선순위 순으로 확인한다 | 이번 주 팀 전체의 완료율을 한눈에 파악한다 |
| **주요 시나리오 2** | 칸반 보드에서 업무를 드래그해 진행 단계를 바꾸거나 담당자를 재배정한다 | 업무 하나를 열어 상태를 "진행 중 → 완료"로 바꾸고 저장한다 | 특정 팀원의 업무 부하가 과중한지 확인하고 팀 리더에게 재배분을 요청한다 |
| **주요 시나리오 3** | 마감 임박·미완료 업무를 필터링해 병목을 파악한다 | 업무에 짧은 메모(진행 상황)를 남긴다 | 월말 보고용으로 완료된 업무 목록을 내보낸다 |
