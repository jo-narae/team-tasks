# Requirements — team-tasks

## 기능 요건 (MVP)

| ID | 이름 | 설명 | 우선순위 |
|----|------|------|---------|
| F-01 | 내 업무 목록 조회 | 로그인 후 기본 화면에서 본인에게 배정된 업무를 우선순위·마감일 순으로 표시한다. 오늘 마감 업무는 강조 표시된다. | 상 |
| F-02 | 업무 상태 변경 | 업무 카드에서 상태를 할 일 → 진행 중 → 완료로 변경한다. 완료는 카드 우측 체크 버튼 1회 클릭으로 처리한다 (클릭 2회 이내). | 상 |
| F-03 | 업무 메모 입력·수정 | 업무 상세 화면에서 진행 현황 메모를 작성·덮어쓰기 저장한다. 저장 후 즉시 목록 화면으로 복귀한다. | 상 |
| F-04 | 신규 배정 업무 식별 | 오늘 새로 배정된 업무를 목록 상단 '오늘 배정됨' 섹션에 분리 표시하고 신규 배지를 붙인다. | 중 |
| F-05 | 미완료 업무 필터 | 오늘 마감 + 미완료 조건으로 목록을 필터링해 잔여 업무를 한눈에 확인한다. | 중 |

## 비기능 요건

| 영역 | 지표 | 목표치 |
|------|------|--------|
| 성능 | 내 업무 목록 최초 로드 시간 (LCP) | ≤ 2 s (3G 환경 기준) |
| 성능 | 상태 변경 후 UI 반영 시간 | ≤ 300 ms |
| 보안 | 인증되지 않은 API 요청 차단율 | 100 % (JWT 검증 실패 시 401 반환) |
| 보안 | 타인 업무 데이터 접근 시도 차단율 | 100 % (본인 소유 리소스만 응답) |
| 가용성 | 월간 서비스 가동률 | ≥ 99.5 % (월 허용 다운타임 ≤ 3.6 h) |
| 접근성 | WCAG 2.1 AA 준수 항목 | 색 대비 ≥ 4.5 : 1, 모든 인터랙티브 요소 키보드 접근 가능 |
